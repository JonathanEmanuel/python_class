# **Temario**

- **Clase 01: Fundamentos y repaso general de Python:** Sintaxis básica; variables y tipos de datos (str, int, float, bool); entrada y salida (input(), print()); conversión de tipos (int(), float()); condicionales y bucles; ejercicios guiados.
[Ver Clase 01](./clase_01/README.md)

- **Clase 02: Listas y funciones simples:** Creación, modificación y recorrido de listas; definición de funciones (parámetros y retorno); práctica tipo examen: Carga y procesamiento de valores en listas.

- **Clase 03: Validación de datos y manejo de entradas del usuario:** Validación de cadenas (longitud, contenido) y números (rangos, no negativos); manejo de errores frecuentes; funciones validadoras reutilizables; simulación de interacciones con el usuario.

- **Clase 04: Operaciones con listas y condiciones más complejas:** Iteraciones con acumuladores y contadores; filtrado y búsqueda en listas; reglas de negocio con condicionales compuestos/anidados (descuentos, multiplicadores); ejercicio integrador "COMÚM o PREMIUM".

- **Clase 5: Simulación completa del examen y práctica final:** Resolución paso a paso de un modelo de examen; buenas prácticas (nombres claros, comentarios, pruebas); repaso de dudas.
