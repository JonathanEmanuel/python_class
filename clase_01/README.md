**Variables y tipos de datos** 
- Las variables almacenan información.
- En Python no es necesario declarar el tipo explícitamente (es de tipado dinámico).

``` py
    nombre = "Julieta"
    edad = 25
    altura = 1.65

    print(nombre)
    print("Edad:", edad)
    print("Altura:", altura)
```
